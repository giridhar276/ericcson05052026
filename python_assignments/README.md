# Python Real-Time Assignments Package

This package contains real-time classroom assignments for Python foundation topics.

## Included

- 13 topic-wise assignment notebooks
- 1 master assignment index notebook
- 3 selected solution notebooks for trainer reference
- employee.csv
- sales.csv

## Topics Covered

1. String methods
2. List methods
3. Tuple methods
4. Dictionary methods
5. Set methods
6. Conditions
7. Loop statements
8. User-defined functions
9. Different ways of passing arguments
10. Lambda functions
11. map() and filter()
12. File handling
13. File handling with context manager

## Suggested Usage

1. Ask students to open one assignment notebook at a time.
2. Let them complete the starter code.
3. Use selected solution notebooks only after students try the assignments.
4. For file handling notebooks, keep employee.csv and sales.csv in the same folder.
